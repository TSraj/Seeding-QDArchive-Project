# MLOps Study
This repository includes the replication package of our grey literature review on MLOps: requirements and architectures.

1. ***MLOPs.atlproj***
   Grounded Theory (codes, groupings and analysis) was performed using the Atlas.ti tool for qualitative data analysis
2. ***MLOpsReq.pdf***
   Mapping between requirements and sources
3. ***Source_Table.pdf***
   A summary and meta-data of sources used   
3. ***GLR_methodology.png***
   The gray literature review methodology flowchart   
4. ***Survey_Answers.xlsx***
   The questionnaire and answers from six MLOps experts
5. ***Interviews***
   The scanned images of the notes taken during the interviews 
7. ***Approval letter ERB2024JADS13.pdf***
   ERB approval letter for interview-based study
